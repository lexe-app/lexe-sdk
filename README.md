# Lexe SDK

Build on [Lexe](https://lexe.app) — self-custodial Lightning wallets powered by
secure enclaves. Each user gets their own always-online node that can send and
receive payments 24/7.

## Getting Started

### Python

```bash
pip install lexe-sdk
```

See the [Python SDK docs](python/) for the full quickstart.

### Rust

```toml
[dependencies]
lexe = "0.1"
```

See the [Rust SDK docs](rust/) for the full quickstart.

## Source Code
