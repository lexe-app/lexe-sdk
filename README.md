# Lexe SDK

The Lexe SDK allows you to programmatically create and control self-custodial,
always-online Lightning wallets with free node hosting.

Read the docs: [docs.lexe.tech](https://docs.lexe.tech/)

## CLI

The Lexe CLI allows you to create and control self-custodial, always-online
Lightning wallets from the command line.

Install:

```bash
curl -fsSL https://lexe.app/install-cli.sh | bash
```

Docs:

- [Overview](https://docs.lexe.tech/cli/)
- [Source code](https://github.com/lexe-app/lexe-public/tree/master/lexe-cli)

## Python

The Python SDK provides a simple interface that makes it easy to integrate a
self-custodial Lightning wallet into any Python application, or create and
manage a wallet interactively via a Python REPL.

Install:

```bash
pip install lexe-sdk
```

Docs:

- [Overview](https://docs.lexe.tech/python/)
- [Quickstart](https://docs.lexe.tech/python/quickstart/)
- [API Reference](https://python.lexe.tech)
- [PyPI](https://pypi.org/project/lexe-sdk/)
- [Source code](https://github.com/lexe-app/lexe-public/tree/master/sdk-uniffi)

## Rust

The Rust SDK is our most advanced SDK, ideal for building secure, reliable,
high-performance applications.

Install:

```bash
cargo add lexe
```

Docs:

- [Overview](https://docs.lexe.tech/rust/)
